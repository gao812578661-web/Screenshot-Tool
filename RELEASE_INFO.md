# RefScrn 发行说明 (Release Info)

RefScrn 是一款基于 C# WPF 开发的轻量级、高性能屏幕截图与标注工具。本版本为稳定发布版。

## ? 核心功能 (Core Features)

1. **精准裁切**：支持自由拖拽选择截图区域，实时显示分辨率标签。
2. **专业标注工具栏**：
   - **矩形/椭圆**：快速框选重点。
   - **箭头**：指引视觉焦点。
   - **画笔/文本**：自由涂鸦或添加文字说明。
   - **马赛克**：一键模糊敏感信息。
3. **先进扩展功能**：
   - **OCR 文字识别**：利用本地算力快速提取图片中的文字。
   - **中英翻译**：集成轻量级翻译引擎，支持识别结果一键翻译。
4. **便捷操作**：
   - **自动保存/剪贴板**：支持一键复制到剪贴板或保存为本地文件。
   - **多级撤销**：记录每一笔标注，随时回退。

## ?? 快捷键说明 (Hotkeys)

- **Alt + A**：进入截图模式（默认全局快捷键）。
- **Esc**：取消当前截图。
- **Enter**：确认当前截图并保存到剪贴板。
- **Ctrl + Z** (截图模式下)：撤销上一步标注。

## ?? 安装与运行 (Installation & Run)

- **绿色单文件**：本程序为 Release 独立 EXE 文件，无需安装 .NET 运行时。
- **运行环境**：Windows 10 / 11 (x64)。

## ? 版本修订记录

- **v1.0.0**：初始版本发布，集成基础标注、OCR 和翻译功能。
- **视觉优化**：工具栏采用圆润圆角设计 (Radius 12)，提升现代感。
